# SW 与 Creo 倒圆角字段映射整理

## 1. 目的

本文单独整理 SolidWorks 与 Creo 两套系统在倒圆角（fillet / round）特征上的：

- 字段级映射关系
- 字段值级映射关系
- 可直接映射、条件映射、不可直接映射的边界

本文目标不是讨论 GUI 操作流程，而是为后续 `CADExchange` 统一数据结构设计、读写桥接和 round-trip 语义保真提供一份可直接引用的映射说明。

---

## 2. 总结结论

SW 与 Creo 的倒圆角不能压缩成单一的：

- `edges + radius`

这种极简结构。

更合理的统一方式是：

1. 先定义一层 `UnifiedFillet`
2. SW / Creo 分别映射到 `UnifiedFillet`
3. 无法进入公共语义的字段保留在 `vendor extension`

建议统一主结构至少覆盖：

- `type`
- `referenceMode`
- `sectionShape`
- `references`
- `radiusItems`
- `isAsymmetric`
- `tangentPropagation`
- `curvatureContinuous`
- `conicValue`

否则会丢失：

- Creo 的 `reference type / through curve / chordal / transition / attach`
- SW 的 `full round / partial edge / keep features / overflow`

---

## 3. 建议统一字段

```cpp
enum class FilletType {
  Constant,
  Variable,
  FullRound,
  FaceFillet,
  ThroughCurve,
  Chordal,
  Unknown
};

enum class ReferenceMode {
  EdgeChain,
  FaceFace,
  FullRoundThreeFaces,
  EdgeSurface,
  EdgeEdge,
  Unknown
};

enum class SectionShape {
  Circular,
  Conic,
  CurvatureContinuous,
  Unknown
};

enum class ConicValueMode {
  None,
  Rho,
  Radius,
  CreoConicValue
};

struct FilletRadiusItem {
  double position;
  double radius1;
  double radius2;
  bool hasRadius2;
};

struct UnifiedFillet {
  FilletType type;
  ReferenceMode referenceMode;
  SectionShape sectionShape;
  ConicValueMode conicValueMode;

  std::vector<RefEntityGroup> references;
  std::vector<FilletRadiusItem> radiusItems;

  bool tangentPropagation;
  bool isAsymmetric;
  bool curvatureContinuous;

  double conicValue;
  bool hasConicValue;

  Json::Value swExtension;
  Json::Value creoExtension;
};
```

## 4. 字段级映射

### 4.1 类型与参考模式

| SW 字段/语义 | Unified 字段 | Creo 字段/语义 | 映射状态 | 说明 |
|---|---|---|---|---|
| `swFeatureFilletType_Simple` | `type = Constant` | `PRO_ROUND_TYPE_CONSTANT` | 可直接映射 | 最稳定公共能力 |
| `swFeatureFilletType_VariableRadius` | `type = Variable` | `PRO_ROUND_TYPE_VARIABLE` | 可直接映射 | 两边都有变半径 |
| `swFeatureFilletType_Face` | `type = FaceFillet` | 常对应 `referenceType = PRO_ROUND_REF_SURF_SURF` | 条件映射 | Creo 没有完全同名一级 type，要结合 reference type |
| `swFeatureFilletType_FullRound` | `type = FullRound` | `PRO_ROUND_TYPE_FULL` | 可直接映射 | 两边都有 full round 语义 |
| 无 | `type = ThroughCurve` | `PRO_ROUND_TYPE_THROUGH_CURVE` | SW 不可直映射 | Creo 专有一级能力 |
| 无 | `type = Chordal` | `PRO_ROUND_TYPE_CHORDAL` | Creo -> Unified 可映射；SW 不可直映射 | 当前统一结构已承接，SW 侧仍无稳定公共对位 |
| 边集 | `referenceMode = EdgeChain` | `PRO_ROUND_REF_EDGE` | 可直接映射 | 最常见 |
| face fillet 的 `set1/set2` | `referenceMode = FaceFace` | `PRO_ROUND_REF_SURF_SURF` | 条件映射 | 语义接近，但 GUI 行为不一定完全一致 |
| full round 的 `set1/center/set2` | `referenceMode = FullRoundThreeFaces` | `PRO_ROUND_TYPE_FULL` + 多参考面 | 条件映射 | 要按角色映射，不是按字段名平铺 |
| 无明确 SW 公共对位 | `referenceMode = EdgeSurface` | `PRO_ROUND_REF_EDGE_SURF` | Creo -> SW 条件降级 | 可能退化成普通 edge fillet 或 face fillet |
| 无明确 SW 公共对位 | `referenceMode = EdgeEdge` | `PRO_ROUND_REF_EDGE_EDGE` | Creo -> SW 条件降级 | 需看具体 full/edge-edge 语义 |

### 4.2 半径与变半径控制点

| SW 字段 | Unified 字段 | Creo 字段 | 映射状态 | 数值映射规则 |
|---|---|---|---|---|
| `DefaultRadius` | `radiusItems[0].radius1` | `leg1.legValue` | 可直接映射 | 先统一单位 |
| `AsymmetricFillet` | `isAsymmetric` | `leg1/leg2` 是否双值 | 条件映射 | Creo 更偏通过双 leg 表达，不一定有独立布尔位 |
| `DefaultDistance` | `radiusItems[0].radius2` | `leg2.legValue` | 条件映射 | 仅非对称/双半径语义下成立 |
| `Location` | `radiusItems[i].position` | `Position` | 可直接映射 | 建议统一成 0..1 边参数 |
| `Radius` | `radiusItems[i].radius1` | `Leg1` | 可直接映射 | 变半径控制点主值 |
| `Distance2` | `radiusItems[i].radius2` | `Leg2` | 条件映射 | 仅双值 variable 语义 |
| 控制点依附 edge | `radiusItems[i].refEdge` | `RefEdge` | 可直接映射 | 需要保持参考边身份 |

### 4.3 截面形状与连续性

| SW 字段/语义 | Unified 字段 | Creo 字段/语义 | 映射状态 | 说明 |
|---|---|---|---|---|
| circular | `sectionShape = Circular` | `PRO_ROUND_CONIC_DISABLE` | 可直接映射 | 普通圆形截面 |
| `ConicTypeForCrossSectionProfile = Rho` | `sectionShape = Conic` | `PRO_ROUND_CONIC_ENABLE` | 条件映射 | 值本身不能盲目直抄 |
| `ConicTypeForCrossSectionProfile = Radius` | `sectionShape = Conic` | `PRO_ROUND_CONIC_ENABLE` | 条件映射 | 需额外保留 value mode |
| `CurvatureContinuous` | `sectionShape = CurvatureContinuous` 或 `curvatureContinuous = true` | `PRO_ROUND_CURV_CONTINUOUS` | 条件映射 | 两边都有类似语义，但实现细节不同 |
| `GetConicRhoOrRadius()` | `conicValue` | `conicValue` | 条件映射 | 必须连同 `conicValueMode` 一起存 |

### 4.4 行为开关与附加选项

| SW 字段 | Unified 字段 | Creo 对应 | 映射状态 | 说明 |
|---|---|---|---|---|
| `PropagateToTangentFaces` | `tangentPropagation` | `autoContinue` 近似 | 条件映射 | 语义相近但不完全等价 |
| `KeepFeatures` | `swExtension.keepFeatures` | 无稳定公共对位 | 不建议公共映射 | SW 偏操作行为 |
| `EnablePartialEdgeParameters` | `swExtension.partialEdge.enabled` | 无稳定公共对位 | 不建议公共映射 | Creo 主结构无同级公共能力 |
| `GetPartialEdgeFilletData` | `swExtension.partialEdge.*` | 无稳定公共对位 | 不建议公共映射 | 只能扩展保存 |
| overflow type | `swExtension.overflowType` | 无稳定公共对位 | 不建议公共映射 | SW 私有 |
| smooth/straight transition | `swExtension.transitionType` | `transition.type/cap` 可部分类比 | 条件映射 | 只能弱映射，不能当完全等价 |

### 4.5 Creo 专有字段

| Creo 字段 | Unified 字段 | SW 对位 | 映射状态 | 说明 |
|---|---|---|---|---|
| `RoundShapeOption = PRO_ROUND_TYPE_THROUGH_CURVE` | `type = ThroughCurve` | 无 | 不可直映射 | 写回 SW 需降级或报 unsupported |
| `RoundShapeOption = PRO_ROUND_TYPE_CHORDAL` | `type = Chordal` | 无稳定公共对位 | Creo -> Unified 可映射；Unified -> SW 不可直映射 | 当前可在 Creo 往返，写回 SW 仍应降级或标 unsupported |
| `conicDepOption` | `creoExtension.conicDepOption` | 无严格对位 | 条件映射 | SW 需借助 conic mode 承接部分语义 |
| `ballSpine` | `creoExtension.ballSpine` | 无严格对位 | 不建议公共映射 | Creo 建模语义更强 |
| `compoundSpine.collection` | `creoExtension.spineRefs` | 无严格对位 | 不建议公共映射 | SW 通常没有同结构 |
| `attachType` | `creoExtension.attachType` | 无 | 不建议公共映射 | Creo 私有 |
| `transition.transType` | `creoExtension.transitionType` | 可弱对应 SW transition | 条件映射 | 不能保证一一等值 |
| `transition.transCap` | `creoExtension.transitionCap` | 无稳定公共对位 | 条件映射 | 只宜扩展保存 |
| `through_curve_featureid` | `creoExtension.throughCurveFeatureId` | 无 | 不可直映射 | SW 无同级公共建模方式 |

## 5. 字段值级映射

### 5.1 主半径与次半径

| 统一语义 | SW 数值来源 | Creo 数值来源 | 映射规则 | 风险 |
|---|---|---|---|---|
| 主半径 | `DefaultRadius` / variable `Radius` | `Leg1` | 统一单位后直接赋值 | 低 |
| 次半径 | `DefaultDistance` / variable `Distance2` | `Leg2` | 统一单位后直接赋值 | 中，需确认当前 shape 支持双值 |

说明：

- SW `DefaultRadius` 对应主半径
- SW `DefaultDistance` 只在非对称语义下成立
- Creo 更偏向 `leg1 / leg2` 表达，而不是单独的 `isAsymmetric` 开关

### 5.2 变半径控制点位置

| 统一语义 | SW 数值来源 | Creo 数值来源 | 映射规则 | 风险 |
|---|---|---|---|---|
| 控制点位置 | `Location` | `Position` | 若边方向一致，`p -> p`；若反向，`p -> 1 - p` | 高 |

说明：

- 建议统一把位置值定义为沿参考边的 `0..1` 参数比例
- 如果边方向未统一，控制点会出现镜像反转

推荐规则：

```cpp
mappedPos = sameEdgeDirection ? srcPos : (1.0 - srcPos);
```

### 5.3 圆锥截面数值

| 统一语义 | SW 数值来源 | Creo 数值来源 | 映射规则 | 风险 |
|---|---|---|---|---|
| 圆锥参数 | `GetConicRhoOrRadius()` | `conicValue` | 必须带 `conicValueMode` 一起映射 | 高 |

说明：

- SW 侧既可能是 `rho`，也可能是 `radius`
- Creo 侧还有 `conicDepOption`
- 因此不能只保留一个裸 `double`

推荐：

- `sectionShape = Conic`
- `conicValue`
- `conicValueMode`
- `creoExtension.conicDepOption`

### 5.4 Full round 参考集

| 统一语义 | SW 数值来源 | Creo 数值来源 | 映射规则 | 风险 |
|---|---|---|---|---|
| 第一侧参考组 | `set1` | side1 role | 按角色映射 | 中 |
| 中间参考组 | `center` | center role | 按角色映射 | 中 |
| 第二侧参考组 | `set2` | side2 role | 按角色映射 | 中 |

说明：

- Full round 的映射要按“角色”而不是按“字段名字面”处理
- 不建议写成：
  - `set1 -> surface1`
  - `center -> surface2`
  - `set2 -> replaceSurface`

因为 Creo 参考字段的实际含义仍依赖 `reference type` 与 `shape`

### 5.5 切线传播与自动延续

| 统一语义 | SW 数值来源 | Creo 数值来源 | 映射规则 | 风险 |
|---|---|---|---|---|
| 切线延展 | `PropagateToTangentFaces` | `autoContinue` | 仅作近似布尔映射 | 中 |

说明：

- 这两个字段可做统一布尔位承接
- 但语义不是严格等价，写文档和代码时都应标记为“近似映射”

## 6. 推荐映射策略

### 6.1 可强映射进入公共结构

- `type = Constant / Variable / FullRound`
- `referenceMode = EdgeChain`
- `radius1 / radius2`
- `position`
- `sectionShape = Circular / Conic / CurvatureContinuous`

### 6.2 可条件映射进入公共结构

- `FaceFillet`
- `FaceFace`
- `FullRoundThreeFaces`
- `tangentPropagation`
- `conicValue`

### 6.3 不建议放入公共主结构

- SW `partial edge`
- SW `keepFeatures`
- SW `overflow`
- Creo `attachType`
- Creo `transition`
- Creo `spine`
- Creo `through curve`
- Creo `chordal`

这些字段建议进入：

- `swExtension`
- `creoExtension`

---

## 7. 推荐的 SW -> Creo 映射规则

### 7.1 Simple

SW：

- `Simple`
- `Asymmetric = false`
- `Circular`

Creo：

- `PRO_ROUND_TYPE_CONSTANT`
- `PRO_ROUND_REF_EDGE`
- `Leg1 = DefaultRadius`

### 7.2 Simple + Asymmetric

SW：

- `DefaultRadius`
- `DefaultDistance`

Creo：

- `Leg1 = DefaultRadius`
- `Leg2 = DefaultDistance`

前提：

- 目标 Creo round shape 支持双 leg 语义

### 7.3 VariableRadius

每个 SW 控制点：

- `Location -> Position`
- `Radius -> Leg1`
- `Distance2 -> Leg2`

Creo：

- `PRO_ROUND_TYPE_VARIABLE`

### 7.4 FullRound

SW：

- `FullRound`
- `set1 / center / set2`

Creo：

- `PRO_ROUND_TYPE_FULL`
- 参考组按 side1 / center / side2 角色映射

### 7.5 FaceFillet

SW：

- `Face`

Creo：

- 优先尝试 `PRO_ROUND_REF_SURF_SURF`

说明：

- 该映射通常只能做到“语义接近”
- 不保证 GUI 行为完全等价

---

## 8. 推荐的 Creo -> SW 映射规则

### 8.1 CONSTANT + EDGE

Creo：

- `PRO_ROUND_TYPE_CONSTANT`
- `PRO_ROUND_REF_EDGE`

SW：

- `Simple`

### 8.2 VARIABLE + EDGE

Creo：

- `PRO_ROUND_TYPE_VARIABLE`
- `PRO_ROUND_REF_EDGE`

SW：

- `VariableRadius`

### 8.3 FULL

Creo：

- `PRO_ROUND_TYPE_FULL`

SW：

- `FullRound`

### 8.4 SURF_SURF

Creo：

- `PRO_ROUND_REF_SURF_SURF`

SW：

- 可尝试 `Face`

说明：

- 这里属于条件映射，不是绝对一一对应

### 8.5 THROUGH_CURVE / CHORDAL

Creo：

- `PRO_ROUND_TYPE_THROUGH_CURVE`
- `PRO_ROUND_TYPE_CHORDAL`

SW：

- 无稳定公共直接对位

建议：

- 降级为普通 fillet，或
- 明确标记 unsupported

---

## 9. 最终判断

SW 与 Creo 的倒圆角：

- 可以建立统一主结构
- 但不能依赖一个完全无扩展字段的纯公共 schema 来无损承接

最小合理建模方式是：

1. 公共层承接：
   - `constant`
   - `variable`
   - `full-round`
   - `asymmetric`
   - `conic`
   - `curvature-continuous`
2. 扩展层承接：
   - SW `partial edge / keep features / overflow`
   - Creo `attach / transition / spine / through-curve / chordal`

如果后续要在 `CADExchange` 中落地统一结构，建议优先把：

- `Constant`
- `Variable`
- `FullRound`
- `EdgeChain`
- `FaceFace`
- `radiusItems`
- `sectionShape`

这组字段先作为第一阶段能力范围。
