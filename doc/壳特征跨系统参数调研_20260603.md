## 1. SolidWorks 壳特征 API 信息

### 1.1 读取侧 (IShellFeatureData)

| 字段 | API 来源 | 类型 | 说明 |
|------|----------|------|------|
| 厚度 | `IShellFeatureData::get_Thickness` | `double` | 壳壁默认厚度 |
| 方向 | `IShellFeatureData::get_Direction` | `long` | `0` = 内侧抽壳 (shell inward), `1` = 外侧抽壳 (shell outward) |
| 移除面 | `IShellFeatureData::get_FacesRemoved` | `VARIANT` (SAFEARRAY of `IFace2`) | 被抽除的面（即壳体开口面） |
| 多厚度面 | `IShellFeatureData::get_MultipleThicknessFaces` | `VARIANT` (SAFEARRAY of `IFace2`) | 具有非默认厚度的面 |
| 多厚度面数量 | `IShellFeatureData::GetMultipleThicknessFacesCount` | `long` | 多厚度面的数量 |
| 多厚度面值 | `IShellFeatureData::GetMultipleThicknessAtIndex` | `double` (per index) | 每个面的独立厚度值 |

**读取前置条件**：必须调用 `IShellFeatureData::AccessSelections(modelDoc, ..., &granted)` 获取选择集访问权，读取完毕后调用 `ReleaseSelectionAccess()` 释放。

### 1.2 创建侧 (IModelDoc2)

| API | 说明 |
|-----|------|
| `InsertFeatureShell(thickness, direction)` | 主创建调用。`direction=VARIANT_FALSE`=内侧, `VARIANT_TRUE`=外侧。需预先将移除面选入 SelectionMgr mark=0/1 |
| `InsertFeatureShellAddThickness(thickness)` | 多厚度面前置调用。需预先将多厚度面选入 SelectionMgr mark=2 |

**创建流程**：
1. 清除选择集
2. 选择移除面（mark=0 首个，mark=1 后续）
3. 如有多厚度面：选择多厚度面（mark=2），调用 `InsertFeatureShellAddThickness`
4. 调用 `InsertFeatureShell(thickness, direction)`

### 1.3 Demo 层提取的信息结构

```cpp
struct CShellInfo {                        // SW demo 层
    double m_thickness;                    // 默认厚度
    bool m_isInnerShell;                   // true=内侧抽壳, false=外侧抽壳
    std::vector<ShellFaceItem> m_facesRemoved;           // 移除面列表 (含 refEntity)
    std::vector<ShellThicknessItem> m_multipleThicknessFaces; // 多厚度面列表 (含 refEntity + thickness)
};
```

## 2. Creo 壳特征 API 信息

### 2.1 特征标识符

- 特征类型：`PRO_FEAT_SHELL`（整型枚举值）
- 元素树构建方式：通过 `ProElementAlloc` + `ProElemtreeElementAdd` 逐级组装，最终由 `ProFeatureWithoptionsCreate` 创建

### 2.2 元素树参数

| 元素 ID | 值类型 | 说明 |
|---------|--------|------|
| `PRO_E_FEATURE_TYPE` | `PRO_VALUE_TYPE_INT` (= PRO_FEAT_SHELL) | 特征类型 |
| `PRO_E_STD_FEATURE_NAME` | `PRO_VALUE_TYPE_WSTRING` | 特征名称 |
| `PRO_E_BODY` | (容器) | 体选择容器 |
| `PRO_E_BODY_USE` | `PRO_VALUE_TYPE_INT` (= PRO_BODY_USE_SELECTED) | 体使用方式 |
| `PRO_E_BODY_SELECT` | `PRO_VALUE_TYPE_SELECTION` | 目标实体（默认体） |
| `PRO_E_SHELL_SRF` | ProReference 数组 | **要移除的表面引用** |
| `PRO_E_SHELL_THICK` | `PRO_VALUE_TYPE_DOUBLE` | **默认壳厚度** |
| `PRO_E_SHELL_FLIP` | `PRO_VALUE_TYPE_INT` (PRO_B_TRUE/PRO_B_FALSE) | **翻转方向**：0=内侧, 1=外侧 |
| `PRO_E_STD_SURF_COLLECTION_APPL` | `ProCollection` | **排除曲面集合** — 从壳特征中排除的表面（不被抽壳影响） |
| `PRO_E_SHELL_LACE_BNDRY` | `PRO_VALUE_TYPE_INT` | 边界处理选项（lace boundary） |
| `PRO_E_SHELL_ALT_CUT_METHOD` | `PRO_VALUE_TYPE_INT` | 交替切割方法 |
| `PRO_E_ST_SHELL_LOCL_LIST` | Array | 多厚度面列表容器 |
| └ `PRO_E_ST_SHELL_LOCL_CMPD` | (容器) | 单个多厚度面条目 |
|   └ `PRO_E_ST_SHELL_SPEC_SRF` | ProReference | **特定面引用** |
|   └ `PRO_E_ST_SHELL_SPEC_THCK` | `PRO_VALUE_TYPE_DOUBLE` | **该面的独立厚度** |

### 2.3 `PRO_E_STD_SURF_COLLECTION_APPL` 详细结构

Creo 的排除曲面集合 (`ExcludeSurfaces`) 使用 `ProSrfcollection` 体系：

```
ProCollection
  └ ProSrfcollinstr (instruction)
      type: PRO_SURFCOLL_SINGLE_SURF (单面选择), PRO_B_TRUE (包含这些面)
      └ ProSrfcollref (per surface)
          type: PRO_SURFCOLL_REF_SINGLE
          reference: ProReference → 具体的表面引用
```

**语义说明**：这是 Creo 壳特征中特有的"排除曲面"概念。用户可以选择某些面不被抽壳影响（保留原位置），与"移除面"是互补概念。

### 2.4 Demo 层提取的信息结构

## 3. 两系统信息归一化分析

### 3.1 直接对等字段

| 归一化概念 | SW 来源 | Creo 来源 | 归一化建议 |
|-----------|---------|-----------|-----------|
| **默认厚度** | `m_thickness` (double) | `shell_thick` (double) | 统一为 `double`，单位跟随模型单位 |
| **方向/翻边** | `m_isInnerShell` (true=内侧) | `shell_flip` (0=内侧, 1=外侧) | 统一为 bool: `true`=内侧抽壳(inward), `false`=外侧抽壳(outward) |
| **移除面列表** | `m_facesRemoved` (vector of face ref) | `shell_srf` (vector of ProReference) | 统一为 `vector<shared_ptr<CRefFace>>` |
| **多厚度面列表** | `m_multipleThicknessFaces` (face ref + thickness) | `shell_locl_cmpd` (face ref + thickness) | 统一为 `vector<{CRefFace, double}>` |

### 3.2 Creo 特有字段

| 字段 | Creo 元素 | 语义 | 归一化策略 |
|------|----------|------|-----------|
| **目标体引用** | `PRO_E_BODY_SELECT` | 被抽壳的实体 | 当前阶段可选：SW 无显式多体概念（隐式操作活动体）。保留为可选字段 `optional<CRefEntityBase>` |
| **排除曲面集合** | `PRO_E_STD_SURF_COLLECTION_APPL` | 从抽壳操作中排除的表面 | 当前阶段可选：SW 无直接对应概念。保留为可选字段 `optional<vector<CRefFace>>` |
| **边界处理** | `PRO_E_SHELL_LACE_BNDRY` | 控制壳边界边处理 | Creo 专有行为选项，不纳入 phase-1 统一结构。后续按需扩展 |
| **交替切割方法** | `PRO_E_SHELL_ALT_CUT_METHOD` | 控制相交曲面切割方式 | Creo 专有行为选项，不纳入 phase-1 统一结构。后续按需扩展 |

### 3.3 SW 特有字段

SW 的创建过程需要区分选择标记（mark=0/1/2）来控制面的角色，但这是创建侧的 API 调用细节，不反映到统一模型中。

## 4. 最小归一化结构（Phase-1）

### 4.1 设计原则

参照已有的 `CFillet`/`CChamfer`/`CRib` 模式：
- 只保留 **跨系统共同的参数语义**
- Creo/SW 专有行为选项放入可选扩展字段
- 不复制 CAD 原生 API 的控制细节

### 4.2 建议的统一结构

```cpp
// --- 枚举 ---

/** @brief 壳特征厚度方向模式 */
enum class ShellThicknessDirection {
  Unknown = 0,
  Inward,   // 内侧抽壳：向实体内部偏移厚度
  Outward   // 外侧抽壳：向实体外部偏移厚度
};

// --- 多厚度面条目 ---

struct CShellThicknessFace {
  std::shared_ptr<CRefFace> face;  ///< 具有独立厚度的面引用
  double thickness = 0.0;           ///< 该面的独立厚度值（跟随模型单位）
};

// --- 壳特征 ---

struct CShell : public CFeatureBase {
  // ---- 核心参数（跨系统共同） ----
  double thickness = 0.0;                                   ///< 默认壳壁厚度
  ShellThicknessDirection direction{ShellThicknessDirection::Unknown}; ///< 抽壳方向

  std::vector<std::shared_ptr<CRefFace>> facesToRemove;     ///< 被移除的面（开口面）
  std::vector<CShellThicknessFace> thicknessFaces;          ///< 非默认厚度的面

  // ---- 系统专有扩展（Phase-1 可选） ----
  std::shared_ptr<CRefEntityBase> targetBody;               ///< 目标实体引用（Creo: PRO_E_BODY_SELECT）
  std::vector<std::shared_ptr<CRefFace>> excludedFaces;     ///< 排除曲面（Creo: PRO_E_STD_SURF_COLLECTION_APPL）

  CShell() { featureType = FeatureType::Shell; }
};
```

### 4.3 FeatureType 枚举扩展

```cpp

  Revolve,

  Fillet,
```

## 5. 与现有特征对比及注意事项

### 5.1 与 Fillet/Chamfer 的相似性

壳特征与倒角/圆角特征都属于 **非草图驱动** 的特征类型（不需要 `profileSketchID`），更接近 `CChamfer` 的模式：
- 直接引用拓扑面（`CRefFace`）
- 不需要 `CProfiledFeatureBase` 基类
- 创建侧通过面选择 + 参数 API 构建

### 5.2 与 ThinWall 的区分

**关键警示**：壳特征 (Shell) 与薄壁 (ThinWall) 是不同概念，务必区分：

| 概念 | 定义 | 使用场景 |
|------|------|---------|
| **ThinWall** (薄壁) | 对 Extrude/Revolve/Sweep 轮廓的法向双侧偏移，形成薄壁实体 | 单一特征创建时的参数选项 |
| **Shell** (壳/抽壳) | 对整个或多实体进行空腔掏空，移除指定面并统一壁厚 | 独立特征，作用于已有实体 |

现有 `CProfiledFeatureBase::thinWall` 和 `ThinWallOption` 已经覆盖了薄壁场景，壳特征应作为独立的 `FeatureType::Shell` 添加，不与 ThinWall 混淆。

### 5.3 SW 类型名称映射

SolidWorks 中壳特征的 `GetTypeName()` 返回值需在实机验证确认。基于 API 推测：
- SW 类型名：`"Shell"` (待实机确认)
- 注册点到 `read_extract.cpp` 的 `BuildDefaultExtractionRegistry()`

### 5.4 Creo 类型码映射

- Creo 特征类型常量：`PRO_FEAT_SHELL`
- 在 `read_to_cadexchange.cpp` 的 type_code 分支中新增处理
