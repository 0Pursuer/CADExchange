## 1. CADExchange 统一结构定义

### 1.1 FeatureType 枚举

```cpp
enum class FeatureType {
  Unknown,   // 0
  Extrude,   // 1
  Revolve,   // 2
  Sweep,     // 3
  Fillet,    // 4
  Chamfer,   // 5
  Rib,       // 6
  Shell,     // 7  ← 新增
  Sketch,    // 8
  DatumPlane // 9
};
```

### 1.2 ShellThicknessDirection 枚举

| 枚举值 | 语义 | SW 对应 | Creo 对应 |
|--------|------|---------|-----------|
| `Unknown` (0) | 未指定 | — | — |
| `Inward`   (1) | **内侧抽壳**：从实体表面向内部偏置厚度，实体被掏空 | SW `Direction=0` / `VARIANT_FALSE` | Creo `PRO_B_TRUE` |
| `Outward`  (2) | **外侧抽壳**：从实体表面向外部扩展厚度，实体被包覆 | SW `Direction=1` / `VARIANT_TRUE`  | Creo `PRO_B_FALSE` |

> ⚠️ **重要**：Creo 的 `PRO_E_SHELL_FLIP` 语义与直觉相反——`PRO_B_TRUE` (1) 表示内侧，`PRO_B_FALSE` (0) 表示外侧。这是因为 Creo 将壳的默认方向定义为外侧，而 "flip" 表示翻转（翻转为内侧）。

### 1.3 CShellThicknessFace 结构

```cpp
struct CShellThicknessFace {
  std::shared_ptr<CRefFace> face;  // 具有独立厚度的面引用
  double thickness = 0.0;           // 该面的独立厚度值（跟随模型单位）
};
```

| 字段 | 说明 | SW 来源 | Creo 来源 |
|------|------|---------|-----------|
| `face` | 拓扑面引用，包含 center/normal/parentFeatureID 等几何指纹 | `MultipleThicknessFaces[i]` (IFace2) | `PRO_E_ST_SHELL_SPEC_SRF` (ProReference) |
| `thickness` | 该面的独立厚度值 | `GetMultipleThicknessAtIndex(i, &thickness)` | `PRO_E_ST_SHELL_SPEC_THCK` (ProElement double) |

### 1.4 CShell 结构（核心）

```cpp
struct CShell : public CFeatureBase {
  double thickness = 0.0;                                    // 默认壳壁厚度
  ShellThicknessDirection direction{Inward};                  // 抽壳方向
  vector<shared_ptr<CRefFace>> facesToRemove;                // 被移除的面
  vector<CShellThicknessFace> thicknessFaces;                // 非默认厚度面
  shared_ptr<CRefEntityBase> targetBody;                     // [可选] 目标实体
  vector<shared_ptr<CRefFace>> excludedFaces;                // [可选] 排除曲面
};
```

| 字段 | 必填 | 说明 |
|------|:---:|------|
| `thickness` | ✅ | 默认壳壁厚度。SW `get_Thickness` / Creo `PRO_E_SHELL_THICK` |
| `direction` | ✅ | 抽壳方向。Inward(内侧) / Outward(外侧) |
| `facesToRemove` | ✅ | 被移除的面（开口面）。每个面是一个 `CRefFace` 引用 |
| `thicknessFaces` | — | 具有非默认厚度的面列表。每个条目包含面引用 + 独立厚度值 |
| `targetBody` | — | 目标实体引用。Creo 特有的 `PRO_E_BODY_SELECT`，SW 无此概念 |
| `excludedFaces` | — | 排除曲面集合。Creo 特有的 `PRO_E_STD_SURF_COLLECTION_APPL` |

## 2. SW ↔ CADExchange 映射

### 2.1 读侧提取 (Extract)

| 步骤 | SW API 调用 | 提取内容 | CADExchange 目标 |
|------|-------------|---------|----------------|
| 1 | `feat->GetDefinition()` → `IShellFeatureData` | 获取壳特征数据接口 | — |
| 2 | `shellData->AccessSelections(modelDoc, …, &granted)` | 获取选择集访问权限 | — |
| 3 | `shellData->get_Thickness(&thickness)` | 默认壁厚 | `CShell::thickness` |
| 4 | `shellData->get_Direction(&swDirection)` | 0=内侧, 1=外侧 | `CShell::direction`（经 `MapSwDirectionToUnified` 转换） |
| 5 | `shellData->get_FacesRemoved(&vFaces)` | 移除面 SAFEARRAY | `CShell::facesToRemove`（每个 IFace2 → `BuildReferenceFromSwDispatch` → `CRefFace`） |
| 6 | `shellData->GetMultipleThicknessFacesCount(&count)` | 多厚度面数量 | — |
| 7 | `shellData->get_MultipleThicknessFaces(&vMultiFaces)` | 多厚度面 SAFEARRAY | — |
| 8 | 对每个多厚度面: `shellData->GetMultipleThicknessAtIndex(i, &t)` | 该面的独立厚度 | `CShellThicknessFace` {face, thickness} |
| 9 | `shellData->ReleaseSelectionAccess()` | 释放访问权 | — |

**方向映射**：
```
SW get_Direction() → long:      0        1
     ↓
MapSwDirectionToUnified:     Inward   Outward
     ↓
CShell::direction:           Inward   Outward
```

### 2.2 写侧创建 (Create)

| 步骤 | SW API 调用 | 参数来源 |
|------|-------------|---------|
| 1 | `modelDoc->ClearSelection2(VARIANT_TRUE)` | 清空选择集 |
| 2 | 选择首个移除面: `SelectReferenceInSw(…, mark=0)` | `CShell::facesToRemove[0]` |
| 3 | 选择后续移除面: `SelectReferenceInSw(…, mark=1)` | `CShell::facesToRemove[1..N]` |
| 4 | 选择多厚度面: `SelectReferenceInSw(…, mark=2)` → 立即调用 `InsertFeatureShellAddThickness(item.thickness)` | `CShell::thicknessFaces[*]` |
| 5 | `modelDoc->InsertFeatureShell(thickness, swDirectionBool)` | `CShell::thickness` + `MapUnifiedDirectionToSwBool(direction)` |

**方向映射**：
```
CShell::direction:       Inward         Outward
     ↓
MapUnifiedDirectionToSwBool: VARIANT_FALSE  VARIANT_TRUE
     ↓
SW API:                  0 (内侧)       1 (外侧)
```

## 3. Creo ↔ CADExchange 映射

### 3.1 读侧提取 (Extract)

#### 元素树结构

```
PRO_E_FEATURE_TREE
├── PRO_E_FEATURE_TYPE     → PRO_FEAT_SHELL
├── PRO_E_STD_FEATURE_NAME
├── PRO_E_BODY
│   ├── PRO_E_BODY_USE     → PRO_BODY_USE_SELECTED
│   └── PRO_E_BODY_SELECT  → ProSelection (目标实体)
├── PRO_E_SHELL_THICK      → double (默认厚度)
├── PRO_E_SHELL_FLIP       → int (PRO_B_TRUE=内侧, PRO_B_FALSE=外侧)
├── PRO_E_SHELL_SRF        → ProArray of ProReference (移除面)
├── PRO_E_ST_SHELL_LOCL_LIST
│   └── PRO_E_ST_SHELL_LOCL_CMPD[0..N]
│       ├── PRO_E_ST_SHELL_SPEC_SRF  → ProReference (特定面)
│       └── PRO_E_ST_SHELL_SPEC_THCK → double (该面厚度)
├── PRO_E_SHELL_LACE_BNDRY
├── PRO_E_SHELL_ALT_CUT_METHOD
└── PRO_E_STD_SURF_COLLECTION_APPL  → ProCollection (排除曲面，暂未采集)
```

#### 提取流程

| 步骤 | 方法 | 提取内容 |
|------|------|---------|
| 1 | `ProFeatureElemtreeExtract()` → 获取完整元素树 | — |
| 2 | `GetElementIntByPath({PRO_E_FEATURE_TYPE})` 判断是否为 `PRO_FEAT_SHELL` | — |
| 3 | `GetElementDoubleByPath({PRO_E_SHELL_THICK})` | → `shell_thickness` |
| 4 | `GetElementIntByPath({PRO_E_SHELL_FLIP})` | → `shell_flip` (PRO_B_TRUE/PRO_B_FALSE) |
| 5 | `ProElemtreeElementVisit()` 递归遍历所有元素 | — |
| 5a | 遇到 `PRO_E_SHELL_SRF` → `ProElementReferencesGet()` 获取 `ProReference[]` → 每个 ref → `ProReferenceToSelection` → `ProSelectionModelitemGet` → id/type | → `shell_remove_faces` |
| 5b | 遇到 `PRO_E_ST_SHELL_LOCL_LIST` → `ProElementArrayGet()` 获取 compound 数组 → 每个 compound → `ProElementChildrenGet` 获取子元素 → `PRO_E_ST_SHELL_SPEC_SRF` (ref) + `PRO_E_ST_SHELL_SPEC_THCK` (double) | → `shell_thickness_faces` |

#### 面引用几何提取 (ConvertProReferenceToMetadata)

| 步骤 | API 调用 | 提取字段 |
|------|---------|---------|
| 1 | `ProReferenceToSelection(ref, &sel)` | 转为 ProSelection |
| 2 | `ProSelectionModelitemGet(sel, &item)` | → `ref_id`, `ref_type` |
| 3 | `ProGeomitemFeatureGet(geomitem, &owner_feat)` | → `ref_owner_feature_id` |
| 4 | `ProGeomitemToSurface(geomitem, &surface)` | 获取表面句柄 |
| 5 | `ProGeomitemdataGet(geomitem, &geom_data)` | 获取表面几何数据(uv_min, uv_max, plane e3) |
| 6 | `ProSurfaceXyzdataEval(surface, uv_mid, center, v1, v2, v3)` | → `ref_center[]` (UV 中点) |
| 7 | 对于平面: `geom_data->data.p_surface_data->srf_shape.plane.e3[]` | → `ref_normal[]` |

**方向映射**：
```
Creo PRO_E_SHELL_FLIP:  PRO_B_TRUE   PRO_B_FALSE
     ↓
                        Inward       Outward
     ↓
CShell::direction:      Inward       Outward
```

### 3.2 写侧创建 (Create)

#### 元素树构建流程

| 步骤 | API | 参数来源 |
|------|-----|---------|
| 1 | `ProElementAlloc(PRO_E_FEATURE_TREE)` | 创建根元素 |
| 2 | `AddElementInt(…, PRO_E_FEATURE_TYPE, PRO_FEAT_SHELL)` | 特征类型 |
| 3 | `ProElementAlloc(PRO_E_BODY)` + `PRO_E_BODY_USE`(PRO_BODY_USE_SELECTED) + `ProSolidDefaultBodyGet` → `ProSelectionAlloc` → `PRO_E_BODY_SELECT` | 目标实体（默认体） |
| 4 | `AddElementDouble(…, PRO_E_SHELL_THICK, thickness)` | `CShell::thickness` |
| 5 | `AddElementInt(…, PRO_E_SHELL_FLIP, flip_value)` | `Inward→PRO_B_TRUE, Outward→PRO_B_FALSE` |
| 6 | `AddElementInt(…, PRO_E_SHELL_LACE_BNDRY, 0)` | 边界处理默认值 |
| 7 | `AddElementInt(…, PRO_E_SHELL_ALT_CUT_METHOD, 0)` | 交替切割方法默认值 |
| 8 | **移除面**: `ProArrayAlloc(ProReference[])` → 每个 face → `ResolveReferenceSelection` 定位 → `ProSelectionToReference` → `ProArrayObjectAdd` → `ProElementReferencesSet(PRO_E_SHELL_SRF)` | `CShell::facesToRemove` |
| 9 | **多厚度面**: `ProElementAlloc(PRO_E_ST_SHELL_LOCL_LIST)` → 每个 item → `ResolveReferenceSelection` 定位面 → 构建 `PRO_E_ST_SHELL_LOCL_CMPD` (含 `PRO_E_ST_SHELL_SPEC_SRF` + `PRO_E_ST_SHELL_SPEC_THCK`) → 添加到 locl_list | `CShell::thicknessFaces` |
| 10 | `ProFeatureWithoptionsCreate(model_sel, tree, PRO_FEAT_CR_INCOMPLETE_FEAT, PRO_REGEN_NO_FLAGS)` | 创建特征 |

#### 面引用定位 (ResolveReferenceSelection)

```
输入: CRefFace { parentFeatureID, center, normal, refType=TOPO_FACE }
     ↓
① 优先: SelectionFromPoint(model, center, PRO_SURFACE)  → 全局坐标搜索
② 备用: TryResolveSurfaceFromFeatureByGeometry(owner_feature_id, ref)
         → 收集特征内所有面 → 评分 (normal_alignment≥0.95 + dist²)
③ 兜底: SelectionFromFeatureGeomitemAtPoint(owner_feature_id, center)
```

## 4. XML 序列化格式

```xml
<Feature ID="FB-3" Name="壳1" Suppressed="false" Type="Shell"
         Thickness="0.01" Direction="Inward">
    <FacesToRemove>
        <FaceRef ParentFeatureID="FB-2" TopologyIndex="0"
                 U="(1,0,0)" V="(0,1,0)" Normal="(0,0,1)"
                 Center="(0.05,0.01,0.1)" Type="Face"/>
    </FacesToRemove>
    <ThicknessFaces>
        <ThicknessFace Thickness="0.005">
            <FaceRef ParentFeatureID="FB-2" TopologyIndex="0"
                     Normal="(0,1,0)" Center="(0.025,0.038,0.067)"/>
        </ThicknessFace>
        <!-- ... -->
    </ThicknessFaces>
    <!-- 可选扩展 -->
    <TargetBody>...</TargetBody>
    <ExcludedFaces>...</ExcludedFaces>
</Feature>
```

| 属性 | 类型 | 说明 |
|------|------|------|
| `Thickness` | double | 默认壳壁厚度，单位跟随模型 UnitSystem |
| `Direction` | string | `"Inward"` 或 `"Outward"` |
| `FacesToRemove/FaceRef` | 引用数组 | 被移除的面，每项含 ParentFeatureID/Center/Normal/U/V |
| `ThicknessFaces/ThicknessFace` | 复合数组 | 非默认厚度面，每项含独立的 `Thickness` 属性 + `FaceRef` |

## 5. 校验规则

| 规则 ID | 严重度 | 条件 |
|---------|:---:|------|
| `SHELL_001` | Error | `thickness <= 0.0` |
| `SHELL_002` | Error | `direction == Unknown` |
| `SHELL_003` | Warn | `facesToRemove` 和 `thicknessFaces` 均为空（可能是空操作） |
| `SHELL_004` | Error | `facesToRemove[i]` 为 null |
| `SHELL_005` | Error | `thicknessFaces[i].face` 为 null |
| `SHELL_006` | Error | `thicknessFaces[i].thickness <= 0.0` |
| `SHELL_007` | Error | `excludedFaces[i]` 为 null |
| `REF_008`~`REF_010` | Warn | 面引用的 parentFeatureID 尚未在特征树中出现（顺序依赖） |
| `SCALE_004` | Warn | 厚度值经过单位转换后超出正常范围（< 1μm 或 > 100m） |
